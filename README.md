# DevOps
Script to configure the Ubuntu Server 20.04 LTS, right after the installation.
## Packages to install
+ Install htop;
+ Install virt-manager;
+ Install Docker;
+ Install Epiphany.
## Everithing else
+ Upgrade the system;
+ Create users;
+ Pull Docker images;
+ Create and configure containers.
## Run the script
``` bash
# Download from git:
git clone https://github.com/Senai-Dev-ops/DevOps.git
# Make it executable
cd DevOps/
chmod +x SetUp.sh
# Execute it
sudo ./SetUp.sh
```
